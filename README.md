# InteractiveExample - Class Presentation Site

This repository now includes a GitHub Pages-ready presentation site generated from the project design rationale.

## Included Files

- `DESIGN_ASPECTS.md` - source design narrative
- `docs/index.html` - public GitHub Pages landing page
- `docs/slides.html` - browser slide deck
- `docs/print.html` - print-optimized slide layout
- `docs/Particle-Artwork-Deck.pdf` - exported PDF deck
- `docs/site.css` - shared styling for both pages

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. Open repository **Settings -> Pages**.
3. Under **Build and deployment**, set:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (or your default branch)
   - **Folder:** `/docs`
4. Save, then wait for deployment.

Your site URL will be:

- `https://<your-username>.github.io/<repo-name>/`

Slide deck direct link:

- `https://<your-username>.github.io/<repo-name>/slides.html`

PDF direct link:

- `https://<your-username>.github.io/<repo-name>/Particle-Artwork-Deck.pdf`

## Presenting the Deck

- Open `slides.html`
- Use keyboard: `Left/Right`, `PageUp/PageDown`, `Home/End`
- Or click `Prev` and `Next`
