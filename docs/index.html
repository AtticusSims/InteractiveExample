Particle Artwork Tool

A design-led interactive tool that turns gesture and facial movement into living particle compositions. This page is the GitHub Pages version of the class documentation and slide presentation.

Why This Tool Exists

The project reframes camera interaction as a design process rather than a filter effect. Users do not just view output; they perform it.

Core intent: create a digital mirror where motion, emotion, and identity become visual language in real time.

Design Process Use

  • Motion sketchbook for ideation and expressive form testing
  • Prototype embodied interaction for communication systems
  • Generate visual references for posters, campaigns, and editorial frames
  • Capture stills and clips as reusable studio source material

Design Thinking Process

  • Intent + audience: playful, critique-friendly, quickly testable
  • Interaction-first: validate hand/face input before polishing visuals
  • Behavior tuning: calibrate glow, density, and color for readable expression
  • Reflection layer: use debug overlays to explain system behavior during critique

Key UI/UX Decisions

  • Two-scene menu: lowers cognitive load and gives immediate direction
  • Dark high-contrast framing: focuses attention on particle motion
  • Hidden camera in main view: supports abstraction over literal self-view
  • Clear gesture-response mapping: makes behavior learnable fast
  • Integrated debug controls: doubles as design explanation tooling

Development Note (Design Emphasis)

Implementation was iterative: interaction prototype -> live testing -> visual tuning -> interface clarity. Technical systems like tracking and simulation support the outcome, but the guiding objective is communicative design through embodied interaction.
